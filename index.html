(h1) Títulos e Subtítulos (h6)

(voltar)

As tags h1 até h6 no HTML são usadas para definir títulos e subtítulos em uma página. Elas indicam a hierarquia dos conteúdos:

h1 → título mais importante (usado geralmente uma vez por página para o título principal).
h2 → subtítulo do h1.
h3 → subtítulo do h2.
h4 → subtítulo do h3.
h5 → subtítulo do h4.
h6 → título de menor importância.

✨ Importância: SEO (Otimização para buscadores): o Google e outros mecanismos de busca usam os títulos para entender a estrutura e relevância da página.

Acessibilidade: leitores de tela usam a hierarquia de títulos para ajudar pessoas com deficiência visual a navegar pelo site.

Organização visual: além do tamanho da fonte que muda automaticamente, a hierarquia ajuda os visitantes a entenderem melhor a estrutura do conteúdo.

👉 Resumindo: h1 a h6 servem para estruturar e organizar títulos e subtítulos em ordem de importância.

Tags e seus conceitos (i, u, strong, p, a, )

(voltar)

🔹 (i) — Itálico

  • Função principal: coloca o texto em itálico.
  • Uso semântico (significado): usado para destacar palavras diferentes do fluxo normal do texto, como:
    1. termos estrangeiros (login, deadline)
    2. nomes de livros, músicas, filmes (O Senhor dos Anéis)
    3. pensamentos ou vozes internas (“preciso estudar mais...”)
  • ⚠️ Importante: visualmente é parecido com (ênfase), mas (em) tem mais valor semântico. (i) é mais visual.

🔹 (u) — Sublinhado

  • Função principal: sublinha o texto.
  • Uso semântico (significado): usado para marcar texto não ortodoxo ou que precisa de atenção especial (ex.: erros em uma prova, nomes próprios em dicionários).
  • Problema comum: links já aparecem sublinhados por padrão, então usar (u) pode confundir o usuário.

🔹 (strong) — Negrito (ênfase forte)

  • Função principal: deixa o texto em negrito
  • Uso semântico (significado): indica importância ou urgência.
  • Diferença para (b): (b) deixa em negrito apenas visualmente. Já (strong) informa ao navegador e ao Google que aquele conteúdo é relevante.
  • Acessibilidade: leitores de tela (para deficientes visuais) podem mudar o tom de voz para destacar.

🔹 (p) — Parágrafo

  • Função principal: cria um bloco de texto separado.
  • Uso semântico (significado): indica que aquele texto é uma ideia ou parágrafo independente.
  • Comportamento padrão: o navegador coloca uma margem automática antes e depois do (p), deixando espaço entre parágrafos
  • Boa prática: sempre use (p) para textos normais. Não é correto apenas quebrar linha com (br) várias vezes.

🔹 (a) — Âncora / Link

  • Função principal: cria links clicáveis.
  • Uso semântico (significado): indica navegação, conexão entre páginas, ou âncoras dentro da própria página.
  • Atributo principal:
    1. href → define o destino do link (pode ser site externo, página interna ou parte da própria página).
  • Outros atributos importantes:
    1. target = "_blank" → abre o link em uma nova aba.
    2. title = "..." → mostra uma dica ao passar o mouse.
    3. download → faz o navegador baixar o arquivo em vez de abrir.

🔹Exemplos:

  • 👉 Link externo:
  • (a href ="https://www.google.com")Ir para o Google(/a)

  • 👉 Link interno (mesma página):
  • (a href ="#contato")Ir para a seção de contato(/a)

  • 👉 Link para download:
  • (a href ="curriculo.pdf" download)Baixar Currículo(/a)

Resumo geral com semântica:

  • (i) → itálico (para termos especiais, estrangeiros, nomes, pensamentos).
  • (u) → sublinhado (para chamar atenção, mas cuidado com links).
  • (strong) → negrito com importância real (SEO e acessibilidade).
  • (p) → cria parágrafo (estrutura e separação de ideias).
  • (a) → cria link (navegação dentro ou fora da página).

Tags o que fazem, quando usar e já está ultrapassado (caso de (font)).

(voltar)

🔹 (ol) — Ordered List (Lista ordenada)

  • Função: cria uma lista numerada ou ordenada.
  • Visual padrão: cada item aparece com números (1, 2, 3...)
  • Uso: quando a ordem dos itens é importante (ex.: receita de bolo, passo a passo, ranking).
  • Atributos importantes:
    1. type → muda o marcador (1, A, a, I, i).
    2. start → define de onde a contagem começa.

🔹 (ul) — Unordered List (Lista não ordenada)

  • Função: cria uma lista com marcadores (bolinhas, quadrados, etc).
  • Uso: quando a ordem não importa (ex.: lista de compras, tópicos soltos).
  • Atributos (quase não usados hoje, mas existem):
    1. type="disc" → bolinha cheia (padrão).
    2. type="circle" → bolinha vazia.
    3. type="square" → quadradinho.

🔹 (li) — List Item (Item da lista)

  • Função: representa cada item dentro de (ol) ou (ul).
  • Uso: sempre deve estar dentro de uma lista ((ol) ou (ul)).
  • Pode conter outras tags dentro: texto, imagens, até listas aninhadas.

🔹 (hr) — Horizontal Rule (Linha horizontal)

  • Função: insere uma linha horizontal na página.
  • Uso semântico (HTML5): representa uma mudança de contexto ou separação de conteúdo.
  • Visual padrão: uma linha cinza que atravessa a tela.
  • Estilização: hoje em dia é comum usar CSS para personalizar (espessura, cor, estilo tracejado).

🔹 (font) — Fonte de texto (⚠️ tag obsoleta)

  • Função (antigamente): mudava a cor, tamanho e tipo de letra do texto.
  • Por que não usar mais: essa tag foi descontinuada no HTML5. Hoje, toda formatação de texto é feita com CSS.
  • Atributos que existiam:
    1. color="red" → definia a cor do texto.
    2. size="5" → mudava o tamanho.
    3. face="Arial" → escolhia a fonte.

✅ Resumo final:

  • (ol) → lista ordenada (com números/letras).
  • (ul) → lista não ordenada (com bolinhas ou marcadores).
  • (li) → item da lista (sempre dentro de (ol) ou (ul)).
  • (hr) → insere uma linha horizontal (separa conteúdo).
  • (font) → (obsoleto) mudava cor/tamanho/fonte → hoje usamos CSS.

Tags e suas Função(sub, sup, mark, del, abbr, blockquote)

(voltar)

🔹(sub) – Texto Subscrito

  • Função: Coloca o texto abaixo da linha normal (subscrito).
  • Uso comum: fórmulas químicas, rodapés ou índices matemáticos

🔹(sup) – Texto Sobrescrito

  • Função: Coloca o texto acima da linha normal (sobrescrito).
  • Uso comum: potências matemáticas, expoentes ou notas de rodapé.

🔹(mark) – Texto Marcado

  • Função: Destaca o texto como se fosse marcado com marca-texto.
  • Uso comum: dar ênfase em palavras ou destacar trechos importantes.

🔹(del) – Texto Deletado

  • Função: Mostra que o texto foi removido ou substituído, ficando riscado.
  • Uso comum: em edições de documentos, preços antigos em promoções.

Exemplo:

  • <p>Preço antigo: <del>R$50,00</del></p>
    • 👉 Resultado:

    Preço antigo: R$50,00

🔹(abbr) - Texto abreviação ou sigla

  • Função: Indica que uma palavra é uma abreviação ou sigla.
  • Uso comum: mostrar o significado quando o usuário passa o mouse (tooltip).

Exemplo:

  • <p>A <abbr title="Organização das Nações Unidas">ONU</abbr> foi criada em 1945.</p>
    • 👉 Resultado:

    ONU (ao passar o mouse mostra “Organização das Nações Unidas”)

🔹(blockquote) – Citação Longa

  • Função: Usada para citações em bloco, geralmente destacadas do texto normal.
  • Uso comum: quando você quer mostrar uma fala, trecho de livro, artigo etc.

Exemplo:

  • <blockquote> "A persistência é o caminho do êxito." – Charles Chaplin </blockquote>
    • 👉 Resultado:

    "A persistência é o caminho do êxito." – Charles Chaplin

📌 Resumo Rápido

  • (sub) → subscrito (abaixo)
  • (sup) → sobrescrito (acima)
  • (mark) → texto marcado
  • (del) → texto deletado (riscado)
  • (abbr) → sigla/abreviação
  • (blockquote) → citação longa
